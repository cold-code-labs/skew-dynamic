# Figure provenance — paper figure → source script → file → evidence

Each manuscript figure is generated by a versioned analysis script and pinned to
an evidence phase in the lineage ledger (`study/lineage.json`,
`docs/LINEAGE.md`). The paper's figure numbers are sequential (Figure 1–18); the
output filenames (`f<N>_*.png`) follow the analysis-block order, so the two
numberings differ from Figure 14 on — this table is the authoritative crosswalk.

Regenerate all figures: `cd study && ./run.sh` (or run the listed script).
