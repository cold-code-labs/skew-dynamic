2. Data — dataset multi-liga, limpeza, recorte ≥2005.
3. Method — retorno ex-post, janela deslizante + pooling, bootstrap, testes.
4. Results

5. Mechanism — cancelamento de caudas; competitividade como propriedade lenta.
6. Robustness — demeaning, overlap, window size, blip forensics.
7. Discussion — implicações; contraste com gain-loss asymmetry financeira.
8. Conclusion.
